Gray Scale Example
==================

This example is transforming a vector with 8bit RGBA color values per channel into gray scale value with the alpha channel.
The example based on [Mattias Kretz talk](https://github.com/CppCon/CppCon2023/blob/main/Presentations/stdsimd_how_to_express_inherent_parallelism_efficiently_via_data_parallel_types.pdf) about expressing parallelism for `std::simd`.
